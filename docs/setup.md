# Setup Guide

## Prerequisites

- Node.js 20+
- pnpm 10+

## Install Dependencies

1. Open the project folder.
2. Run:

```bash
pnpm install
```

## Start Development

Run:

```bash
pnpm run dev
```

The site starts in local development mode with hot module replacement.

## Build Production Files

Run:

```bash
pnpm run build
```

Build output is generated in ./dist.

## Preview Production Build

Run:

```bash
pnpm run preview
```
