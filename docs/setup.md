# Setup Guide

## Prerequisites

- Node.js 20+
- npm 10+

## Install Dependencies

1. Open the project folder.
2. Run:

```bash
npm install
```

## Start Development

Run:

```bash
npm run dev
```

The site starts in local development mode with hot module replacement.

## Build Production Files

Run:

```bash
npm run build
```

Build output is generated in ./dist.

## Preview Production Build

Run:

```bash
npm run preview
```
